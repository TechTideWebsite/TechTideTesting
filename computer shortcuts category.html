Computer shortcuts
Here are some windows shortcuts

Win + D: Show or hide the desktop.

Win + E: Open File Explorer.

Win + L: Lock your PC.

Win + M: Minimize all windows.

Win + R: Open the Run dialog box.

Win + S: Open Search.

Win + I: Open Settings.

Win + X: Open the Quick Link menu (Start menu options like Device Manager, Task Manager, etc.).

Window Management: Win + Left Arrow: Snap the window to the left side of the screen.

Win + Right Arrow: Snap the window to the right side of the screen.

Win + Up Arrow: Maximize the window.

Win + Down Arrow: Minimize the window or restore if maximized.

Win + Shift + Left/Right Arrow: Move the current window to another monitor (if using multiple monitors).

Win + Home: Minimize all windows except the active one.

Win + Tab: Open Task View (shows all your open desktops and windows).

Alt + Tab: Switch between open apps.

Virtual Desktops: Win + Ctrl + D: Add a new virtual desktop.

Win + Ctrl + Left/Right Arrow: Switch between virtual desktops.

Win + Ctrl + F4: Close the current virtual desktop.

Taskbar Shortcuts: Win + [Number]: Open the app pinned to the taskbar in that position (e.g., Win + 1 opens the first pinned app).

Win + T: Focus on the taskbar and cycle through apps.

System & Miscellaneous: Win + P: Open the projection options (for multiple monitors).

Win + U: Open Ease of Access settings.

Win + V: Open Clipboard history (if enabled in settings).

Win + Shift + S: Open Snipping Tool to capture part of your screen.

Win + Pause/Break: Open the System Properties window.

Win + + (plus): Zoom in (Windows Magnifier).

Win + - (minus): Zoom out (Windows Magnifier).

Screenshot Shortcuts: Win + PrtScn: Capture the entire screen and save the screenshot to your Pictures folder.

Win + Shift + S: Open Snip & Sketch to take a screenshot.

Alt + PrtScn: Capture the current window and copy it to the clipboard.

Cortana (if enabled): Win + C: Open Cortana for voice commands (if enabled).

Win + Q: Open Cortana's search box.
