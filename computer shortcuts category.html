Computer shortcuts
We use computer shortcuts to speed up any task we do and they're very useful! here are some shortcuts to hopefully speed up your tasks with just a few keyboard keys! make sure you hold down the keys to execute the action, don't press both at the same time only hold the first then second then even the third key if it says about a third key then it will execute properly

Is there any specific category of shortcuts you'd like to visit? General Windows Shortcuts | Windows Management | Virtual Desktops | Taskbar & System | Screenshots | Miscellaneous
General Windows Shortcuts
Shortcut Action
Win + D Show or hide the desktop
Win + E Open File Explorer
Win + L Lock your PC
Win + M Minimize all windows
Win + R Open the Run dialog box
Win + S Open Search
Win + I Open Settings
Win + X Open the Quick Link menu (Start menu options)
Window Management
Shortcut Action
Win + Left Arrow Snap window to the left side of the screen
Win + Right Arrow Snap window to the right side of the screen
Win + Up Arrow Maximize the window
Win + Down Arrow Minimize or restore window
Win + Shift + Left/Right Arrow Move window to another monitor
Win + Home Minimize all windows except the active one
Win + Tab Open Task View (shows all desktops & windows)
Alt + Tab Switch between open apps
Virtual Desktops
Shortcut Action
Win + Ctrl + D Create a new virtual desktop
Win + Ctrl + Left/Right Arrow Switch between virtual desktops
Win + Ctrl + F4 Close the current virtual desktop
Taskbar & System
Shortcut Action
Win + [Number] Open the pinned app at that position (Win + 1 for first app, etc.)
Win + T Focus on the taskbar and cycle through apps
Win + P Open projection options
Win + U Open Ease of Access settings
Win + V Open Clipboard history (if enabled)
Win + Pause/Break Open System Properties
Screenshots
Shortcut Action
Win + PrtScn Capture full screen and save to Pictures folder
Win + Shift + S Open Snipping Tool
Alt + PrtScn Capture active window and copy to clipboard
